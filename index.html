01

확장 프로그램 목적

Peaches & Cream은 SillyTavern 위에서 동작하는 팝업 앱입니다. 채팅 자체를 건드리지 않고, 캐릭터·유저 정보를 저장하거나 AI로 다양한 콘텐츠를 생성해 롤플레이 경험을 풍부하게 만들어줍니다.

💾
정보 저장 & 프롬프트 반영
유저와 캐릭터의 신체·성향 정보를 저장하고, 저장된 데이터를 AI 프롬프트에 자동으로 주입해 생성 품질을 높입니다.
✨
AI 콘텐츠 생성
채팅 맥락을 읽어 후기, 꿈 일기, 반성문, 독백 등 다양한 창작 콘텐츠를 자동으로 생성합니다.
🎮
미니 게임 & 인터랙션
Red Flag 퀴즈, Clinic Q&A 등 채팅 외의 인터랙티브한 경험을 제공합니다.
⚡
빠른 액션 태그
채팅창 하단 툴바에서 태그 하나로 원하는 행동을 AI에게 즉시 지시할 수 있습니다.
02

메인 앱 4종

홈 화면 APPS 섹션에 있는 4가지 앱입니다. 어떤 캐릭터와 대화 중이든 공통으로 사용할 수 있습니다.

👤
Profile
유저 & 캐릭터 신체 정보 관리

유저와 캐릭터의 신체 정보, 신체 특징, 성감대 등을 직접 입력해두면 확장 프로그램이 해당 내용을 AI 프롬프트에 자동으로 반영해 더 일관된 묘사를 만들어냅니다.

✦ 모든 항목은 반드시 영어로 입력해주세요. 한국어로 입력하면 AI가 제대로 이해하지 못해 묘사 품질이 낮아질 수 있습니다.

  • User 탭 — 유저의 신체 묘사, 특징적인 마크, 성감대를 입력합니다.
  • Character 탭 — 캐릭터의 신체 묘사, 특징, 성감대를 입력합니다.
  • 저장 버튼을 누르면 즉시 프롬프트에 반영됩니다.
입력 예시 (User · Body)
Slender build with a small waist and soft curves. Medium-sized breasts with light pink nipples. Smooth fair skin that flushes easily when aroused. A small mole below the left collarbone.
🚩
Red Flag
캐릭터 행동 판정 퀴즈

AI가 현재 캐릭터의 행동이나 습관을 카드 형태로 3개 생성합니다. 유저는 각 카드에 대해 "🚩 레드 플래그" 또는 "😍 오히려 좋아"를 선택하고, 선택 후에는 캐릭터의 실제 반응도 AI가 생성해줍니다.

  • 캐릭터 정보를 바탕으로 AI가 카드를 생성합니다.
  • 3장이 끝나면 레드 플래그 / 오히려 좋아 집계 결과를 보여줍니다.
  • 매번 새로운 카드가 생성되어 중복이 없습니다.
🩺
Clinic
가상 산부인과 Q&A

성적인 주제에 대해 궁금한 점을 입력하면, 개성 있는 가상의 산부인과 의사가 답변해줍니다. 의사는 매번 랜덤으로 배정되며 각자 다른 말투와 성격을 가지고 있습니다.

  • Dr. Harrison (엄격), Dr. Emily Chen (놀란), Dr. Wallace (무덤덤), Dr. Park (열정적) 중 랜덤 배정
  • 자주 묻는 질문 빠른 선택 버튼 제공 (거친 행위, 수갑, 애프터케어 등)
  • 질문 입력 후 Enter 또는 버튼으로 바로 답변 받기 가능
📝
Sorry Not Sorry
반성문 & 탄원서 생성기

캐릭터가 잘못한 내용을 입력하면 공식 서식의 반성문 또는 탄원서를 AI가 생성합니다. 유머러스하지만 격식 있는 문서 형식으로 출력됩니다.

  • Apology 탭 — 반성문. 잘못, 변명, 다짐 섹션으로 구성된 공식 문서 형식
  • Petition 탭 — 탄원서. 피청원인·청원인·사유 포함, 도장 스타일 스탬프 포함
  • 생성 전 어떤 행동에 대한 것인지 간단히 입력할 수 있습니다.
03

Vault — 캐릭터 전용 공간

현재 대화 중인 캐릭터에 종속된 공간입니다. Vault 안의 앱들은 해당 캐릭터의 채팅 내용과 설정을 바탕으로 콘텐츠를 생성합니다. 캐릭터가 바뀌면 Vault의 내용도 달라집니다.

C
Character's Vault
캐릭터 전용 공간 · 8가지 앱
⭐ Reviews
캐릭터에 대한 별점 후기를 AI로 생성하고 누적 저장합니다. 평균 평점과 추천 비율도 자동 집계됩니다.
💬 Off the Record
Caught Off Guard — 들켰을 때 캐릭터의 솔직한 반응 카드
Dark Thoughts — 캐릭터가 속으로 품은 어두운 생각 카드
🌐 World Feed
X(트위터) 형태의 SNS 피드를 생성합니다. Vault Settings에서 세계관 등장인물을 추가하면 해당 인물들과 캐릭터의 상호작용 포스트가 함께 생성됩니다.
🌙 Dream Log
캐릭터가 꾼 꿈을 AI로 생성합니다. NSFW, 코믹, 이별, 싸움, 트라우마 등 7가지 타입 중 가중치 랜덤으로 결정됩니다.
📂 Blackbox
Threat Mail — 캐릭터 앞으로 온 협박 편지 생성
Complaints — 캐릭터에 대한 민원·불만 사례 생성
🎯 Wanted
캐릭터의 죄목과 현상금을 생성하는 수배 시스템입니다. 재판 판결문(무죄/유죄)도 AI로 생성됩니다.
✍️ Monologue
이별, 기억, 만약이라면 등 감성적인 주제로 캐릭터의 1인칭 독백을 생성합니다. 매일 2개씩 새 주제가 제시됩니다.
⚙️ Settings
Vault 내 앱 설정을 관리합니다. World Feed의 세계관 그룹명, Dream Log 설정 등을 변경할 수 있습니다.

커스텀 태그 (예시)
+ 내 태그 추가
1
툴바 켜기
Peaches & Cream 팝업 홈 화면에서 NSFW Toolbar 토글을 ON으로 설정합니다. 채팅창 하단에 즉시 태그 바가 나타납니다.
2
태그 탭해서 사용 — 자유로운 다중 선택
원하는 태그를 탭하면 선택(활성화)되고, 다시 탭하면 해제됩니다. SFW 포함 모든 그룹에서 제한 없이 다중 선택이 가능합니다. 같은 그룹 안에서 여러 개를 고를 수도 있고, 서로 다른 그룹에서 각각 골라 조합할 수도 있습니다. 예를 들어 SFW에서 Kiss + Hug, Position에서 Missionary, Action에서 Slow를 동시에 선택해 한 번에 전송하는 식으로 조합이 무한정 가능합니다. 선택을 모두 취소하려면 초기화 버튼을 누르세요.
3
삽입하려면 Position + Penetrate 조합
삽입 행위를 원할 경우 Position 그룹에서 원하는 자세를 선택하고, Action 그룹에서 Penetrate를 함께 선택해야 합니다. Penetrate 단독으로도 사용 가능하지만, 자세를 함께 지정하면 더 구체적인 묘사가 생성됩니다.
4
Condom ON / OFF
툴바 상단의 Condom ON 버튼을 활성화하면 삽입 시 AI가 자동으로 콘돔을 착용하는 묘사를 넣습니다. Condom OFF로 설정하면 콘돔 없이 진행하는 묘사가 생성됩니다. 아무것도 선택하지 않으면 콘돔 관련 묘사는 생략됩니다. Condom 설정은 Apply 후에도 유지되며, 다음 액션에도 계속 적용됩니다.
5
커스텀 태그 추가 — 반드시 영어로
홈 화면에서 Toolbar 태그 관리로 이동하면 각 그룹에 태그를 추가하거나 삭제할 수 있습니다. 커스텀 태그는 반드시 영어로 입력해야 합니다. 한국어나 다른 언어로 입력하면 AI가 올바르게 인식하지 못할 수 있습니다. 회색으로 표시된 기본 태그는 삭제가 불가합니다.
6
태그 전체 초기화
Toolbar 태그 관리 화면 하단의 커스텀 태그 전체 초기화 버튼으로 추가한 커스텀 태그를 모두 삭제할 수 있습니다.

토큰 이용 비용

Peaches & Cream의 AI 생성 기능은 SillyTavern에 연결된 API를 그대로 사용합니다. 아래는 Gemini 3.1 Pro Preview 기준 참고 가격입니다.

구분 200K 토큰 이하 200K 토큰 초과
입력 (Input) $2.00 / 1M 토큰 $4.00 / 1M 토큰
출력 (Output) $12.00 / 1M 토큰 $18.00 / 1M 토큰
💡
실제 사용 비용은 매우 적습니다
앱 하나가 호출하는 토큰은 대부분 수백~수천 토큰 수준입니다. 1M 토큰 = 약 75만 단어 분량이므로, 일반적인 개인 사용에서는 비용이 거의 발생하지 않습니다. 채팅 자체의 토큰 사용량과 합산되어 청구된다는 점은 참고해 두세요.
⚠️
Preview 모델이라 가격이 변동될 수 있습니다
Gemini 3.1 Pro Preview는 정식 출시 전 모델로, 향후 가격이 변경될 수 있습니다. 최신 가격은 Google AI 공식 가격 페이지에서 확인하세요.
06

알아두어야 할 점

사용 전에 미리 알아두면 좋은 제한 사항과 알려진 이슈입니다.

⚠️
메인 API에서만 작동합니다
NSFW Toolbar는 채팅 입력창과 직접 연계되는 방식으로 동작하기 때문에, 현재 메인 API 연결 상태에서만 정상 작동합니다.
🖥️
콘솔에 POST 오류가 떠도 무시하세요
브라우저 개발자 도구 콘솔에 POST ... 형태의 오류 메시지가 표시될 수 있습니다. 이는 확장 프로그램의 내부 통신 과정에서 발생하는 것으로, 실제 사용에는 영향이 없습니다. 안심하고 무시하셔도 됩니다.
🐢
오류가 쌓이면 느려질 수 있어요
위 POST 오류가 장시간 사용 중 누적되면 브라우저가 조금씩 느려질 수 있습니다. 다음 두 가지 방법으로 간단히 초기화됩니다.
  • 페이지 새로고침 — 오류 로그가 모두 초기화됩니다.
  • 다른 캐릭터로 전환 — 전환 시점에 자동으로 초기화됩니다.
장시간 같은 캐릭터로 사용할 때는 가끔 새로고침해 주는 것을 권장합니다.
